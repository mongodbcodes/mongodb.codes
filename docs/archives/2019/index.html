Features

FTS - Full Text Search

- General speed improvements that speedup search up to 20x times.
- Negative terms filtered by index.
- Positive phrase with multiple words perform AND operation on each word in phrase.
- Huge memory handeling improvements.
- Score searches return results in order by default.
- No need for extra sorting and fast limit score based searches.
- See roadmap here

$cluster - group by distance

It's aggregator pipeline stage that generate buckets based on distance between values.

- You define distance and all data will be grouped based on distance between variables
- it's possible to do grouping on multiple values like 2d Geo location
- you can define grouping expressions to define output
